	        <span  itemprop="articleBody text"><section class="page">
<p>If company CEOs can be <a href="http://www.computerworld.com/article/2913387/it-management/think-your-management-job-is-safe-beware-the-iceo.html">replaced</a> by computers, why not the president of the United States? That, in fact, is exactly what's been proposed for IBM's Watson.</p>
<p>Watson has already made quite a name for itself, of course, dating back to its 2011 <a href="http://www.pcworld.com/article/219893/ibm_watson_vanquishes_human_jeopardy_foes.html">victory</a> on Jeopardy. Since then, it's gone on to make <a href="http://www.cio.com/article/2955053/using-watson-ibm-and-cvs-aim-to-predict-when-your-health-is-about-to-decline.html">medical predictions</a>, <a href="http://www.computerworld.com/article/2949756/emerging-technology/ibm-watsons-next-trick-helping-you-write-better.html">critique</a> written text and offer <a href="http://www.pcworld.com/article/3011944/why-is-your-new-product-failing-just-ask-watson-analytics.html">insight</a> on product performance, among other accomplishments.</p><div id="gpt-showcasemobile-2" class="articleBloxAd side-promo side-module side-ad mobile-only smartphone"> </div>
<p>Now, a group called The Watson 2016 Foundation -- which "has no affiliation with IBM," according to its <a href="http://watson2016.com/">Watson for President</a> website -- has emerged to help promote the cognitive computer's presidential prospects.</p>
<p>Watson was designed to compete against humans by understanding a question, using key information to analyze vast volumes of data and then deriving the best possible answers, the group notes. "This is a task all politicians undergo on a daily basis, including the president, and could be more suitably and efficiently executed by an artificial intelligence," Watson for President says.</p>
<p>"Watson’s capabilities for parallel processing on a given query allow it to evaluate concerns on a variety of metrics, gauging the most beneficial response across all areas," the website says. "Watson can advise the best options for any given decision based on its impact on the global economy, the environment, education, health care, foreign policy, and civil liberties."</p>
<p>Ready to welcome this new AI overlord? You can keep your wallet in your pocket for now: The Watson 2016 Foundation is not accepting donations at this time and does not expect to in the foreseeable future.</p>
<p>"IBM made 12 billion dollars last year, so contributions from the public are not required for the continued research on Watson," the website says.</p>
<p>People who want to support Watson are encouraged to donate to the Electronic Frontier Foundation instead.</p><aside class="nativo-promo smartphone tablet desktop" id=""> </aside>
<p>In the meantime, while it's clearly a stunt, the effort also offers some food for thought.</p>
<p>"Just in the same way humans have facial expressions to convey emotions, Watson changes its visual form to express its level of confidence in a selected answer," the foundation's website notes. "Wouldn’t the country be better if all politicians were that transparent?"</p>
<p>So, would you vote for Watson? You can share your view in a poll now under way on the Watson for President website.</p>
	</section></span>

function selectTSOAds(ads,max) {
	if (ads.length <= max) {
		return ads;
	} else {
		var uniq = {};
		var found = 0;
		var selectedAds = [];
		while (found < max) {
			var ad = ads[Math.floor(Math.random()*ads.length)];
			if (uniq[ad.id] != null) {
				continue;
			} else {
				uniq[ad.id] = true;
				found++;
				selectedAds.push(ad);
			}
		}
		return selectedAds;
	}
	return null;
}
